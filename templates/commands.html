Главная Об авторе О работе Команды

Шпаргалка по командам

Просмотр логов сервиса

Позволяет в реальном времени видеть, что происходит с твоим сайтом и почему он может не запускаться.

sudo journalctl -u service -f

Перезапуск сервиса

Выполнить после ручного редактирования файлов на сервере.

sudo systemctl restart service

Посмотреть ВСЕ активные сервисы

Эта команда выведет список всех сервисов, которые работают в данный момент

systemctl list-units --type=service --state=running

Выключить навсегда (убрать из автозагрузки)

Эта команда сделает так, чтобы сервис не запускался сам при включении приставки

sudo systemctl disable service
